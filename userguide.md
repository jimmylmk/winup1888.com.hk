## 1. 登入後台

1. 在瀏覽器中輸入您的網站後台網址（預設為 `/admin`），例如：
   ```text
   http://localhost:3000/admin
   ```
2. 輸入您的管理員帳號密碼進行登入：
   - **預設帳號**：`admin@accoladegroup.com.hk`
   - **預設密碼**：`adminpassword123` *(建議首次登入後在「Users」選單內修改密碼)*

## 2. 系統設定（Site Settings）

點擊左側導航的 **「系統設定」 > 「Site Settings」**，即可進入網站全局設定區。

### 公司基本資料與聯絡方式
* **Company Name (公司名稱)**：編輯網站上顯示的公司法定名稱（例如：`卓遠集團 ACCOLADE`）。
* **Logo Text**：網頁頂部或標題顯示的 Logo 識別文字。
* **Email**：全局主要聯絡電子郵件。
* **Phone**：全局主要聯絡電話（例如：`+852 3521 2888`）。
* **WhatsApp**：當用戶點擊 WhatsApp 圖示時跳轉的行動連結（格式：`https://wa.me/852XXXXXXXX`）。
* **WhatsApp Display**：在網頁頂部文字顯示的 WhatsApp 號碼。
* **License Text / License Link**：修改公司牌照編號（如 `牌照編號:TC000447`）與證照 PDF 的下載連結。

### 多國辦公室地址管理
網站頁尾（Footer）的各辦事處資訊在此統一進行管理：
* **HK Address**：香港總部地址。
* **SZ Address**：深圳辦事處地址。
* **SZ Phone / SZ WhatsApp**：深圳辦事處專用電話與 WhatsApp 連結。
* **Beijing Address**：北京辦事處地址.
* **Singapore Address**：新加坡辦事處地址.

> [!TIP]
> 以上地址皆支援多語言（繁體中文、簡體中文、英文），請切換後台右上角的語系標籤以進行不同語言的地址翻譯填寫。

### 三層式主導航選單（Navigation Menu）
網站的主導航選單已完全動態化，支援最多 **3 層嵌套**。

* **選單層級結構**：
  * **第一層 (Level 1)**：主選單項目（如「公司註冊」）。
  * **第二層 (Level 2)**：子選單項目（如「香港公司註冊」），在 `subMenuItems` 中新增。
  * **第三層 (Level 3)**：巢狀選單項目（如「註冊香港公司收費」），在 `nestedMenuItems` 中新增。

* **如何設定選單連結 (Link)**：
  - **內部頁面**：輸入相對路徑（例如：關於我們輸入 `/about-us`，香港公司註冊輸入 `/services/hk-company`）。系統會自動在前端補充語系首碼（如 `/en/about-us`）。
  - **外部連結**：直接輸入完整的 URL（如 `https://google.com`）。
  - **頁面錨點**：如需連至首頁的 FAQ 區，輸入 `#faq` 即可。

### 全局 UI 標籤文字
此區塊可以調整全站固定的 UI Wording，如：
- `Price Label` (預設：`服務費用`)
- `Service Cta Fallback` (預設：`立即聯絡顧問諮詢`)
- `Back To Home Label` (預設：`返回首頁`)
- `Inquire More Label` (預設：`查詢更多`)

## 3. 頁面內容管理（Pages）

點擊左側 **「內容管理」 > 「Pages」**。您會看到現有的頁面清單（如 `about-us`、`inquiry`）。

### 編輯一般頁面
* **Title (標題)**：頁面名稱。
* **Slug (網址路徑)**：此頁面的網址識別字（如 `about-us`）。*注意：請勿隨意修改已上線頁面的 Slug，以免連結失效。*
* **Short Description**：頁面的簡短介紹。
* **Content (內容)**：使用富文本編輯器（Lexical Editor）撰寫頁面內文，支援標題（H2、H3）、粗體、清單項目、超連結等。

### 自訂頁面徽章與特色清單
任何頁面都可以啟用左右排版的「徽章與特色列表」：
* **Badge (頁面徽章)**：例如 `專業顧問對接`，會以綠色小膠囊標籤的形式呈現在頁面頂部。
* **Features (特色清單)**：可新增多筆特色，每筆包含：
  - **Title**：特色標題。
  - **Description**：特色詳細說明。
  - **Icon (圖示)**：可選 `Check (打勾)`、`Users (用戶)`、`Dollar (貨幣)`。

### 聯絡詢價頁面專屬設定（Inquiry Form Settings）
當頁面的 Slug 是 `inquiry` 時，後台會**自動顯示**聯絡詢價表單的專屬設定區：

* **表單標題與描述**：自訂表單左側展示的文字標題與導言。
* **欄位標籤與 Placeholder (預留字)**：
  - 可以完全自訂 姓名、電子郵件、聯絡電話、諮詢服務、留言訊息 的標籤文字與輸入框提示文字（例如將 `Your Name` 改為 `您的姓名`）。
* **表單送出狀態文字**：自訂按鈕文字（如 `提交詢價需求`）與送出中文字（如 `正在提交...`）。
* **送出成功畫面**：自訂送出成功的標題、說明內容，以及返回首頁的按鈕文字。
* **Service Hints (服務動態提示文字)**：
  - 您可以為不同的服務配置專屬提示。當用戶在表單的服務下拉選單中選擇了該服務時，表單下方會自動顯示該段提示文字。
  - **Service Slug**：輸入要關聯的服務路徑（例如：`hk-company`）。
  - **Hint Text**：輸入提示內容（例如：`提示：香港公司最快可在 1 天內完成註冊...`）。

## 4. 服務項目管理（Services）

點擊左側 **「Services」** 即可編輯所有在網頁上展示的服務大類（例如：`hk-company` 香港公司註冊）。

### 新增/修改服務
* **Title (服務名稱)**：例如 `香港公司註冊及成立`。
* **Slug**：服務的網址（如 `hk-company`）。
* **Icon**：選擇代表該服務的圖示。
* **Price (參考價格)**：輸入數字，例如 `3800`。*注意：若此處留空，網頁上會自動改為顯示「請諮詢我們」。*
* **Content**：該項服務的詳細介紹內文。

### 設定自訂價格後綴與按鈕 (CTA)
* **Price Suffix (自訂價格後綴)**：
  - 預設價格後面會顯示 `+` 符號（如 `HK$ 3,800+`）。
  - 如果您想顯示特別週期或文字，可填寫此欄。例如填寫 `起/年`，前台會顯示 `HK$ 3,800 起/年`。
* **CTA Text (自訂按鈕文字)**：
  - 預設聯絡按鈕顯示全局設定的「立即聯絡顧問諮詢」。
  - 若您想針對這項服務使用不同的按鈕文字，可填寫此欄（例如：`免費獲取公司註冊方案`）。

## 5. 多語言編輯技巧

本系統支援繁體中文（`zh-HK`）、簡體中文（`zh-CN`）和英文（`en`）三種語言。

1. **切換語系**：在編輯任何文檔時，畫面**右上角**會有一個語系下拉選單（預設顯示 `zh-HK`）。
2. **翻譯欄位**：
   - 當您切換語系後，所有**支援多語言**的輸入框（欄位旁通常會有語言小標記）內容會切換為該語系的版本。
   - 請依次切換為 `zh-CN` 與 `en` 並輸入對應的翻譯。
3. **儲存修改**：在填寫完各語系的內容後，點擊右下角的 **「Save」** 或 **「Publish」** 進行發佈。
